# kotlin-racingcar-realtime

## 1단계 요구사항
- [x] 각 자동차를 입력 받는다.
- [x] 목표 거리를 입력 받는다.
- [x] 각 자동차는 각각의 코루틴에서 독립적으로 이동한다.
- [x] 이동은 0ms ~ 500ms 사이의 랜덤한 시간 동안 delay한 후, 1칸 전진한다.
- [x] 이동할 때마다 출력한다.

## 2단계 요구사항
- [ ] 엔터를 입력하면 모든 코루틴이 중단된다.
- [ ] 차량명을 추가할 수 있다. (잘못된 입력은 바로 예외처리)
- [ ] 다시 엔터를 입력하면 코루틴이 제개된다.
